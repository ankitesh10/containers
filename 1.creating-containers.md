# Creating Containers

## Running Images

```sh
docker ps
```

## Download Ubuntu Image

```sh
docker run -it --name docker-host --rm --privileged ubuntu:noble
```

- `-it`: make it interactive.
- `--rm`: when stopped, throw everything away, like logs.
- `--privileged`: gives root privilege. Normally do not use it.
- `noble`: name of the current Ubuntu version.

## Know Version Of Ubuntu

```sh
cat /etc/issue
```

## Make New Folder

```sh
mkdir my-new-root
```

## Create New File

```sh
echo "my super secret thing" >> my-new-root/secret.txt
```

## Change Root

```sh
chroot /my-new-root bash
```

## Get Bash To New Root

Get bash dependencies:

```sh
ldd /bin/bash
```

Now copy all dependencies to the new root, and then copy bash.

## Make New Lib Directories

```sh
mkdir my-new-root/lib{,64}
```

This creates `lib` and `lib64` directories. The first value is an empty string and the second value is `64`.

Example:

```sh
cp /lib/ld-linux-aarch64.so.1 /my-new-root/lib/
```

Common binaries:

- `bash`: `/bin/bash`
- `ls`: `/bin/ls`
- `cat`: `/bin/cat`
