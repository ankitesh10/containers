# Docker Features

## Docker Scout

Docker Scout scans for security vulnerabilities.

```sh
docker scout quickview alpine:3.23.3
```

## Bind Mount

Use local files in containers.

```sh
docker run --mount type=bind,source="$(pwd)"/dist,target=/usr/share/nginx/html -p 8080:80 nginx:latest
```

## Volumes

State that you want to survive between runs can be shared across Docker containers.

## In `docker-volume` Directory

```sh
docker build -t incrementor .
docker run --rm incrementor
```

## Run With Volume For Persistent State

```sh
docker run --rm --env DATA_PATH=/data/num.txt --mount type=volume,src=incrementor-data,target=/data incrementor
```
