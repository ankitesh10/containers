# Docker Image

## Run Hollywood Container

```sh
docker run --rm -it bcbcarl/hollywood
```

`--rm` tells Docker to remove the container when it exits.

## Run In Background

```sh
docker run -dit --name hw --rm bcbcarl/hollywood hw
```

`-dit` runs the container in the background.

## Pause

```sh
docker pause hw
```

## Unpause

```sh
docker unpause hw
```

## Kill

```sh
docker kill hw
```

## Execute And Check Process

```sh
docker exec hw ps aux
```

- `exec`: execute a command in this container.
- `hw`: container name.
- `ps aux`: command to run in this container.

## Attach

Attach to the process. Usually, you want to use `exec` instead of `attach`.

```sh
docker attach hw
```

## Image History

```sh
docker history node
```

`node` is the image name.

## Docker Info

```sh
docker info
```

## Process Inside Container

```sh
docker top hw
```

## Remove Container

```sh
docker rm <container_name>
```

## Show Images

```sh
docker images
```

## Delete Image

```sh
docker rmi <image_name>
```

## Search Docker Images

```sh
docker search js
```

## Help

```sh
docker --help
```
